# Driver Drowsiness Detection System

AI-powered Driver Drowsiness Detection project built using Python, OpenCV, MediaPipe, and Pygame.

## Features
- Real-time face detection
- Eye closure detection
- Drowsiness alert system
- Head movement detection
- Alarm sound alert
- Screenshot capture during drowsiness
- FPS display
- Real-time monitoring

## Technologies Used
- Python
- OpenCV
- MediaPipe
- Pygame

## How to Run

```bash
pip install -r requirements.txt
python appdd.py
```

## Controls
- Press `q` to quit

## Project Output
Detects driver drowsiness and gives alert when:
- Eyes remain closed for several seconds
- Head goes down
