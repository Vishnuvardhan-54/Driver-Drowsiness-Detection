
🚗 Driver Drowsiness Detection System using AI

An AI-powered Driver Monitoring System developed using Python, OpenCV, MediaPipe, and Streamlit that detects driver drowsiness in real time through webcam analysis.

The system continuously monitors the driver's eyes, facial landmarks, and head movements to identify signs of fatigue and alert the driver instantly.

✨ Features

✅ Real-time Face Detection
✅ Eye Closure Detection
✅ AI Face Mesh Tracking
✅ Head Pose Detection
✅ Drowsiness Alert System
✅ Live Webcam Monitoring
✅ Automatic Screenshot Capture
✅ Real-time Status Monitoring
✅ Streamlit Web Interface

🛠️ Technologies Used
Python
OpenCV

🚀 How to Run the Project
1️⃣ Clone the Repository
git clone https://github.com/Vishnuvardhan-54/Driver-Drowsiness-Detection.git
2️⃣ Open Project Folder
cd Driver-Drowsiness-Detection
3️⃣ Install Required Libraries
pip install -r requirements.txt
4️⃣ Run the Streamlit App
streamlit run streamlit_app.py
📷 Important Note

⚠️ Webcam detection works only in local system environment.

Browser-based deployment platforms may have limited webcam support.
